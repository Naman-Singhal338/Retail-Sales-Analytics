# Retail Sales Analytics (SQL + Python + Tableau)

End-to-end data analytics project analyzing 4 years of retail sales data (2014–2017) across the US, covering data cleaning, SQL analysis, Python EDA, and an interactive Tableau dashboard — built to answer real business questions about sales, profit, and customer behavior.

## 🎯 Business Objectives
- [ ] Identify sales and profit trends over time
- [ ] Determine which categories/regions/segments drive profitability
- [ ] Quantify the impact of discounting on profit margins
- [ ] Identify top and bottom performing products/customers
- [ ] Recommend data-driven business actions

## 📊 Dataset
- **Source:** Sample Superstore Dataset
- **Size:** 9,994 rows × 21 columns
- **Time Period:** Jan 2014 – Dec 2017
- **Granularity:** One row per order line-item

## 🛠️ Tech Stack
SQL · Python (Pandas, NumPy, Matplotlib, Seaborn) · Jupyter Notebook · Tableau · Git/GitHub
